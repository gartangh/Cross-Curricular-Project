edit hostname with:
	sudo nano /etc/hostname
change raspberrypi to RaspberryPiZeroW and confirm with CTRL+X.

edit interfaces with:
	sudo nano /etc/network/interfaces
change the bottom of the file so it looks like this:
	source-directory /etc/network/interfaces.d

	auto lo
	iface lo inet loopback
	
	allow-hotplug eth0
	iface eth0 inet manual

	allow-hotplug wlan0
	iface wlan0 inet dhcp
		wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf

	allow-hotplug wlan1
	iface wlan1 inet dhcp
		wpa-conf /etc/wpa_supplicant/wpa_supplicant.conf
confirm with CTRL+X.

edit interface eth0:
	sudo nano etc/dhcpcd.conf
add this to the bottom of the file:
	interface eth0
	static ip_address=10.0.0.253/24
	static routers=192.168.1.1
	static domain_name_servers=192.168.1.1
(on your computer, set the IPv4 address to 10.0.0.254)
	
edit wireless connections:
	sudo nano et/wpa_supplicant/wpa_supplicant.conf
change the bottom of the file so it looks like this:
	network={
        ssid="iVisitor"
        psk="WelcomeATimec"
        key_mgmt=WPA-PSK
	}
confirm with CTRL+X.
(you can add more networks yourself)

reboot with:
	sudo shutdown -r now
